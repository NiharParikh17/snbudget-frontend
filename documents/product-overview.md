# Product Overview

## Vision

**SNBudget** helps people understand and control their money by combining
**personal budgeting** with **shared-expense splitting** in one place. Users
should never have to reconcile two apps (a budget tracker + Splitwise) again.

## Target users

- Individuals who want category-based budgets (groceries, rent, fun, etc.).
- Roommates, couples, friend groups, and travel companions who routinely split
  costs and want each person's share to flow into their own budget.

## Core feature pillars

1. **Budgets** — users define categories and monthly (or custom-period) limits,
   and see real-time progress against them.
2. **Transactions** — users record expenses and income, categorized, optionally
   tied to an account.
3. **Splits** — any transaction can be split between SNBudget users using
   shares, percentages, or exact amounts. Each participant's share posts to
   their own budget automatically.
4. **Balances** — running "who owes whom" view across all shared expenses, with
   the ability to settle up.
5. **Insights** — summaries and trends over time.

## Non-goals (for now)

- Direct bank/CC integrations (Plaid, etc.) — manual or CSV import only at
  first.
- Multi-currency conversion. Single currency per user initially.
- Native mobile apps. Web-first; mobile can wrap later.

## Success criteria (early)

- A user can create a budget, add a transaction, split it with another user,
  and see both their budget and the shared balance update correctly — without
  leaving the app.
